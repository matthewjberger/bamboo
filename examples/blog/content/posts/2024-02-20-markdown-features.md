Bamboo supports standard markdown features.

## Text Formatting

You can use **bold**, *italic*, and ~~strikethrough~~ text.

## Lists

Unordered:
- Item one
- Item two
- Item three

Ordered:
1. First
2. Second
3. Third

## Task Lists

- [x] Completed task
- [ ] Incomplete task

## Blockquotes

> This is a blockquote.
> It can span multiple lines.

## Code

Inline `code` works too.
