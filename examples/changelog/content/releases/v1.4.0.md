## New Features

### Pagination Helpers

New helper methods for paginated endpoints:

```javascript
// Iterate through all pages automatically
for await (const user of client.users.listAll()) {
  console.log(user.name);
}

// Or get a specific page
const page = await client.users.list({ page: 2, perPage: 50 });
```

### Custom Fetch Implementation

You can now provide your own fetch implementation:

```javascript
import { AcmeClient } from '@acme/sdk';

const client = new AcmeClient('xxx', {
  fetch: customFetchImplementation,
});
```
