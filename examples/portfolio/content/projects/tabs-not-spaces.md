Static analysis tool for enforcing tabs (the objectively correct choice). Saving bytes one indentation at a time.

## Overview

A command-line tool that enforces consistent indentation using tabs across your entire codebase. Tabs are objectively superior for accessibility and file size.

## Key Features

- Fast static analysis
- Configurable rules
- CI/CD integration
- Auto-fix capability
