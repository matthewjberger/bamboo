Decentralized peer-to-peer network protocol for the new internet. Making centralized servers obsolete.

## Overview

PiperNet is a fully decentralized peer-to-peer network that leverages middle-out compression to create a faster, more private, and truly decentralized internet infrastructure.

## Key Features

- True decentralization with no central servers
- Built-in encryption for privacy
- Leverages middle-out compression for efficiency
- Resistant to censorship and single points of failure
